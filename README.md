
## Запуск

**Сервер:**
```bash
python3 randevu.py
```

**Клиенты:**
```bash
python3 client.py <server_ip>
```

## Важно

- Запускать клиенты быстро друг за другом
- Точный тайминг у меня КРАЙНЕ критичен

## Сетевой трафик

1. **Регистрация** - клиенты отправляют `register` на сервер
2. **Обмен адресами** - сервер отправляет `peer` с адресами клиентов  
3. **NAT пробивание** - клиенты обмениваются `punch` пакетами
4. **P2P чат** - прямое соединение, сообщения идут напрямую
